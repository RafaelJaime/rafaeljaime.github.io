## Description

MatiMola.es is a website built with Vite as a professional portfolio and landing page. The main goal of the project is to present services and generate leads through a clear, fast, conversion-oriented page.

The site includes a contact form that sends the submitted content by email using Formizee as the form handling service.

### Main Features

- Conversion-focused landing page
- Portfolio section to showcase services and experience
- Contact form with email delivery powered by Formizee
- Responsive design for mobile and desktop
- Fast loading and optimized frontend structure with Vite

### Technologies Used

- Vite
- HTML, CSS, and JavaScript
- Formizee
- Responsive design
- Contact forms
- Frontend optimization
