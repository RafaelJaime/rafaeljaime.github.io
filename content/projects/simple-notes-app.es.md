## Descripción

Este proyecto comenzó como una aplicación de notas simples con operaciones CRUD y una integración inicial con **ChatGPT API** para enriquecer la experiencia de creación y edición.

Con el tiempo, la idea evolucionó: se añadió una versión para **tvOS**, adaptando la interfaz y la navegación para Apple TV, manteniendo el enfoque principal en la gestión de notas (crear, leer, actualizar y eliminar).

### Tecnologías utilizadas

- Swift
- SwiftUI
- Xcode
- ChatGPT API (OpenAI)
- Arquitectura CRUD para notas
- Desarrollo multiplataforma Apple (iOS + tvOS)

### Evolución del proyecto

- **Fase 1 (iOS) — 23 Jan 2026:** App de notas simples con CRUD e integración de IA.
- **Fase 2 (tvOS) — 23 Mar 2026:** Versión específica para Apple TV orientada también al CRUD de notas, con UX adaptada al entorno de tvOS.
