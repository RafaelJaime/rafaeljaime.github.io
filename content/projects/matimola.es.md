## Descripción

MatiMola.es es una web desarrollada con Vite como portfolio y landing page profesional. El objetivo principal del proyecto es presentar servicios y generar contactos de clientes potenciales a través de una página clara, rápida y orientada a la conversión.

El sitio incluye un formulario de contacto que envía por correo el contenido escrito por el usuario, utilizando Formizee como servicio de gestión del formulario.

### Características principales

- Landing page enfocada en conversión
- Portfolio para mostrar servicios y experiencia
- Formulario de contacto con envío de correo mediante Formizee
- Diseño responsive adaptado a móvil y escritorio
- Carga rápida y estructura optimizada con Vite

### Tecnologías utilizadas

- Vite
- HTML, CSS y JavaScript
- Formizee
- Diseño responsive
- Formularios de contacto
- Optimización de frontend
