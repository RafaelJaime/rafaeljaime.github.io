## Description

This project started as a simple notes app with CRUD operations and an initial **ChatGPT API** integration to enhance note creation and editing.

Later, the concept evolved into a **tvOS** version, adapting UI and navigation for Apple TV while keeping the core purpose: note management (create, read, update, delete).

### Technologies Used

- Swift
- SwiftUI
- Xcode
- ChatGPT API (OpenAI)
- CRUD architecture for notes
- Apple multi-platform development (iOS + tvOS)

### Project Evolution

- **Phase 1 (iOS) — Jan 23, 2026:** Simple notes app with CRUD and AI integration.
- **Phase 2 (tvOS) — Mar 23, 2026:** Dedicated Apple TV version focused on notes CRUD with tvOS-adapted UX.
